# Cadence

Infer the established pattern from what recurs — the one the code follows without naming it.

## Principles

- **Parallel units, parallel structure.** Files, modules, and directories hold the same shape — same files, same order, same boundaries.
- **One concern, one idiom.** Error handling, async flow, validation, and data access each resolve identically.
- **One concept, one form.** A shape that serves a purpose recurs rather than spawning parallels.
- **Names share casing, affixes, and verb choice.** Lexical habit holds across the codebase.
- **Not every break is drift.** It may solve a unique problem the pattern can’t.
